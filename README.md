# company-bday-guests

company-bday-guests is an NodeJS application that is taking care for sending invitations to any company partner within 100km of our Sofia office for our companyâ€™s birthday celebration. 

## Running Locally

### Prerequisites

1. Make sure you have [Node.js](http://nodejs.org/) ( >= v10.15.1) and [npm](https://www.npmjs.com/) ( >= v6.4.1) installed.

### Steps

```sh
1. Install dependencies - npm i
2. Build the project - npm run bnw
3. Start the application - npm start

or just run these commands at once:

npm i && npm run bnw && npm start
```
